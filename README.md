# Langkah - langkah Setup System
1. Clone Repository
2. Buat Table terlebih dahulu, Query ada di Folder Database
3. Ganti Connection String untuk CustomerService dan SalesOrderService di appsettings.json. Sesuaikan dengan connection database anda.
